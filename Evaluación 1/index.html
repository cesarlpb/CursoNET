    • CSS

      • Cascade StyleSheets, son las hojas de estilo que podemos utilizar en nuestros proyectos con HTML5 y web apps para dar estilo a nuestras páginas. Su versión actual es CSS3.

    • Javascript
    • Bootstrap
    • Visual Code
